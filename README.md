# zip_c

## ZipCompressor: English
  Is a modern, responsive web application that allows you to bundle multiple files into a .zip package 100% locally and securely. <br/> Processing runs entirely within the user\'s browser, ensuring absolute privacy without uploading any data to external servers. <br/> The app features a dynamic real-time preview of the final file size based on the chosen compression level (No compression, Normal, or Maximum), an automatic bilingual interface (PT/EN).

<hr/>

### ZipCompressor: Português
  É um aplicativo web moderno e responsivo que permite empacotar múltiplos arquivos no formato .zip de forma 100% local e segura. <br/> O processamento é feito inteiramente no navegador do usuário, garantindo privacidade absoluta sem o upload de dados para servidores externos. <br/> O app conta com pré-visualização dinâmica do tamanho final do arquivo com base na taxa de compressão escolhida (Sem compressão, Média ou Máxima), interface bilíngue automática (PT/EN).
